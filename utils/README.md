## Init

```bash
# create package.json
$ npm init -y
```

## Setup package.json

```json
{
  ...
  "main": "dist/index.js",
  "types": "dist/index.d.ts",
  "files": ["/dist"],
}
```

## Install

```bash
# npm packages
- typescript

# install
$ npm install typescript
```

## Init TypeScript

```bash
$ npx tsc --init
```
